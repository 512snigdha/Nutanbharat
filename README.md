# Nutanbharat Initiative

A campaign initiative to request the President of India to declare the Cow (Gau Mata) as the National Animal of India.

## Overview

This project provides a platform to support the Nutanbharat initiative, which advocates for the recognition of the Cow as the National Animal of India.

## Building and Running

### Option 1: Open HTML Directly
Simply open the `index.html` file in your web browser.

```bash
# On Linux/Mac
xdg-open index.html

# On Windows
start index.html
```

### Option 2: Using a Local Server

Using Python's built-in server:

```bash
# Python 3
python3 -m http.server 8000

# Then open: http://localhost:8000
```

### Option 3: Using Node.js

```bash
npm install -g http-server
http-server
```
